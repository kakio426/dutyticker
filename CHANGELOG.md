# Changelog

All notable changes to this project will be documented in this file.

## [0.2.0] - 2025-12-30

### Added
- **Ticker Schedule**: Implemented auto-scrolling marquee for the Today's Schedule with pause-on-hover.
- **Premium Broadcast Sound**: Added a new G-Major arpeggio chime for teacher broadcasts.
- **Broadcast Settings**: Added a toggle for broadcast sounds in both the broadcast modal and the Admin page.
- **Persistent Settings**: Broadcast sound preferences are now saved to local storage.
- **Role Status Toggling**: Added ability to check/uncheck roles directly from the dashboard.
- **Enhanced Booster Mode**: Added dramatic visual effects (screen shake, heart pulse, burning fuse) to the 30s booster timer.
- **Theme Visual Optimization**: Updated Nature and Ocean themes with high-contrast complementary colors for better visibility.

### Changed
- Refined Admin UI layout, moving theme selection to the bottom and integrating role randomization.
- Improved card background consistency in the Timeline view.
- Updated Booster button design to match theme aesthetics.

---

## [0.1.0] - 2025-12-28
- Initial release of DutyTicker.
- Basic role management and student assignment.
- Theme engine with Midnight, Nature, Ocean, and Pastel themes.
- Basic timer and broadcast functionality.
